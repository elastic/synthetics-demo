# Synthetics Examples

This repository contains an example Elastic Synthetics project, and some ancillary files for working with it.

Start with the `demo` directory, which was generated by running the `npx @elastic/synthetics init .` command.

The tests suites contained in each on all PRs and merges. See the `.github/workflows` directory to see a sample CI configuration. This is also created
by that same command.

For deploying via Heartbeat see the `heartbeat` folder. Though it is now recommended to use the `npx @elastic/synthetics push` command 
for deploying browser based monitors via kibana
